# moover: Convert Movement Data into Behaviour Workflows

`moover` helps animal scientists move from raw accelerometer data to
canonical movement tables, epoch features, trained models, optimisation
outputs, exported bundles, and predictions on new datasets.

## Details

The package supports both guided beginner workflows and fully scripted,
reproducible runs from JSON specs.
